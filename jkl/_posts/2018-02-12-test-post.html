    • installing 

      sudo apt-get install ruby ruby-dev make gc
sudo gem install jekyll bundler

      mkdir dfeich.github.io
cd dfeich.github.io/

      I then created the jekyll source structure 

      jekyll new jkl-src

      this required typing in my password, because jekyll seemingly pulled in some more ruby gems, and since I am using a centrally installed ruby, the paths to the install area requires root access. Not ideal.

      now, a first test view of the site can be gained by attaching a browser to http://localhost:4000/ after running 

      cd jkl-src
jekyll serve
